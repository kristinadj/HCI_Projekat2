Adding new Species

ID Textbox* in the Add Species Window

This is a textbox that requires input of species' ID.

User types ID manually in the textbox.

This ID defines species internally in the application. This ID has to be unique.


*textbox = blank space where text can be typed in



Add Name Textbox in the Add Species Window

This is a textbox* that requires input of species' name.

Name represents species' name as it is in the real world. Example: Tiger, Dolphin, Tomato etc.

There can be multiple names in the tables of species, but every species must have unique ID in the application.


*textbox = blank space where text can be typed in



Add Description Textbox in the Add Species Window

This is a textbox* that requires input of species' description.

Description contains details about the species.


*textbox = blank space where text can be typed in

Select type of specie

While adding a new species, user can choose a type of species from the combobox (menu).

User can only choose those types that are already added before from the main menu -> Types of species (Only existing types will be offered in the combobox).

If combobox is empty, user can't add a new species, because there are no defined types.



Status of Endangerment

Status of Endangerment Combobox allows user to choose the type of endangerment for the species user wants to add.

User can choose one status of endangerment:

  • Critically Endangered
  • Endangered
  • Vulnerable
  • Dependent on the habitat
  • Close to risk
  • Smallest risk
  • Isolated
  • Partially habituate
  • Habituated


If user selects wrong status of endangerment - status can be changed later in the edit species window.



IUCN List

User can check if the species is on the IUCN list or not (checked for yes, unchecked for no).

The IUCN Red List of Threatened Species (also known as the IUCN Red List or Red Data List), founded in 1965, has evolved to become the world's most comprehensive inventory of the global conservation status of biological species.
It uses a set of criteria to evaluate the extinction risk of thousands of species and subspecies. These criteria are relevant to all species and all regions of the world.
With its strong scientific base, the IUCN Red List is recognized as the most authoritative guide to the status of biological diversity.
A series of Regional Red List are produced by countries or organizations, which assess the risk of extinction to species within a political management unit.



Tourist Status Combobox

User can choose tourist status from the combobox (menu).

Tourist status defines if species lives in the presence of humans.

  • Isolated - Species lives isolated from the humans.
  • Partially Habituate - Species lives isolated in some areas, but can be found partially habituated in humans' presence.
  • Habituate - Species lives entirelly in humans' presence.


Income From Tourists

This is a textbox* that requires amount of money species' is generating from tourists.

This amount is displayed in dollars.

Tourists can pay to see some species, and the amount of generated money is typed into this textbox.


*textbox = blank space where text can be typed in



Date of Discovery

This is a textbox* that requires a date of discovery of the species.

User can manually type in the date of discovery.
Format for the date is: month/day/year
Example: 6/29/2018 (29th of June, 2018)

User can also choose a specific date from the calendar shown in the right corner of the textbox.


*textbox = blank space where text can be typed in
