Main Window of the Application

After starting the application, main window will be shown on the desktop.
From this window user can navigate to other windows and operate.

Content:

  • MenuBar
    • Help
    • Tutorial
  • Toolbar
    • Species Card
      • Table of Species
      • Right Click Actions
      • Maps
      • Putting Species on Map
    • Types of Species
      • Table of Types
    • Tags

In the menubar we can find Help and Tutorial actions.

Help

Inside the Help - user can check "Tips".
This action will show more details about species on the map to the user.

Tutorial

After clicking tutorial action - tutorial video will start playing.
This video is intended to show user how to use application's functionalities in the right way.


Toolbar is located just below the menubar.
There are three cards in the menubar:

  • Species Card
  • Types of species Card
  • Tags Card

Species Card

This is the main card. User can search species and filter species from here. Also, user can see the list of all species in the table of species.
Below the filter option there is "On map" functionality which can be checked or unchecked.

Checked - User will search/filter species that are currently on the maps.
Unchecked - User will search/filter species that are not placed on the map.

Table of Species

Tables of species can be found below search/filter/on map functionalities.
This is the table that contains slots of species that are added. Every slot contains an image of a species, ID, and the name.

Right Click Actions:

User can right click on the slot of a species and choose between three options:

  • Details - User can see full details about a selected species.
  • Edit Specie - User can edit informations about a selected species.
  • Delete Specie - User can PERMANENTLY delete a species from the application.

User is intended to put species on the map from the tables of species.

Maps

Four maps are located on the right side of the main window. User can put a species on a chosen map.
Maps are being shown on four cards above the map image. With clicking on one of these cards - user can choose maps.

After selecting a map, species can be put onto it.

Putting a Species on Map

Instructions on putting a species onto map:

  1. Hold left mouse click on species slot at tables of species.
  2. Drag that slot onto selected map.
  3. Release click.

Moving species through maps is allowed.
User can right click on an image of a species which is put on map. After right-clicking, two options can be chosen:

  • Cut
  • Remove
Cut - Species is being saved, so it can be moved to another map: Click on another map-card -> right click on the map -> Paste. Species is now moved to another map.
Remove - Species is being permanently deleted from the application.

Maps are not zoomed out. User can use two given scrolls to navigate on the map. Map itself can't be zoomed out or zoomed in.

Types of Species

This is the second card on the main window out of three main cards.
User can add new type of species if navigating to this card.

Table of Types

Already added types of species are located on the left side of the window, in the table of types.
This table contains slots of already added types.

Adding New Type

On the right side of the window, there is a form of adding a new type of species.
Full information about a species type consists of label, name, description and an image. All of these informations must be inserted if the user wants to add a new type.
